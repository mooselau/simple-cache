# simple-cache
simple cache library implemented by java

## Upgrading Flow
Step 1. branch out to feature/xxx and make the changes

Step 2. PR to develop branch and waiting the snapshot build, e.g. 1.0-snapshot

Step 3. cut release branch from develop branch, modify the version and remove snapshot, waiting release build, e.g. 1.0 or 1.0-release

Step 4. Develop branch rolls-up to next build with snapshot, e.g. 2.0-snapshot
